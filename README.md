# Embedded AI Coding Rule Skill

## English
A global safety skill for AI-assisted coding, especially embedded and firmware work. It requires discussion before code changes, hardware-first troubleshooting for waveform and data issues, and git checkpoints before large edits.

## 中文
这是一个面向 AI 辅助编程的全局安全技能，尤其适用于嵌入式和固件开发。它要求在改代码前先讨论，在波形和数据异常时先排查硬件，并在大改前先做 Git 检查点。

## 日本語
これは AI 支援コーディング向けのグローバル安全スキルで、特に組み込み開発とファームウェア作業向けです。コード変更前の合意、波形やデータ異常時のハードウェア優先切り分け、大規模変更前の Git チェックポイントを必須にします。
